# bookshelf

## Features
* 首頁搜尋書名：即時顯示搜尋結果
* 切換顯示模式：Gallery圖庫、Table表格
* 返回頂部按鈕：流暢捲動至頁面頂部
* 設定追蹤清單：書本可任意加入追蹤，如欲取消追蹤將彈出確認視窗
* 切換多國語系：中文、英文
